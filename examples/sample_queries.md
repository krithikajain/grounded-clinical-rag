# Example Queries

Below are sample queries that demonstrate retrieval and generation behavior.

## 1. Symptom Retrieval
Which notes mention cough and fever?

## 2. Risk Factor Search
Which patients have hypertension?

## 3. Negation Handling
Which notes explicitly deny diabetes?

## 4. Timeline-Based Query
Which notes describe exertional chest pain?

## 5. Infectious Disease Mention
Is COVID mentioned in any patient record?

## 6. Record-Level Summary
Summarize the symptoms for the patient with note_id trec-cds-2014-4.
